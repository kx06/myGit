# myGit

My implementation of git made using Python

commands working so far :

1. init
2. hash-object
3. cat-file
4. write-tree
5. read-tree
6. commit
7. checkout

I shall update the readme as I add features :)

## Installation Instructions :

```
$ git clone git@github.com:acmpesuecc/myGit.git
$ cd myGit
$ pip install --editable .
```

### Usage :

```
$ mygit --help
```
